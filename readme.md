# Income Calculator (Python)

This is a simple command-line income and expense calculator written in Python.  
It asks the user for monthly income amounts and common expenses, performs calculations, and shows the remaining money after all costs are deducted.

This project demonstrates:
- Python input handling  
- Basic arithmetic operations  
- Helper functions  
- Clean money formatting using f-strings  
- Simple financial logic and calculations  

---

## üöÄ Features

- Prompts for two incomes (your paycheck + partner paycheck)  
- Asks for essential monthly expenses:  
  - Rent  
  - Electricity  
  - Phone bills  
  - Groceries  
  - Loan payments (2)  
- Automatically includes fixed monthly costs:  
  - Chiropractor visits  
  - Car wash subscription  
  - Two YouTube Music subscriptions  
- Displays the remaining balance after all expenses  
- Formats money with commas and two decimals
